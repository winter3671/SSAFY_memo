## 백준 25083번: 새싹
### 문제

아래 예제와 같이 새싹을 출력하시오.

### 입력

입력은 없다.

### 출력

새싹을 출력한다.

### 예제 입력

### 예제 출력
```python
'''

         ,r'"7
r`-_   ,'  ,/
 \. ". L_r'
   `~\/
      |
      |
'''
```

### 내가 작성한 코드
```python
print('''
         ,r'"7
r`-_   ,'  ,/
 \\. ". L_r'
   `~\\/
      |
      |
''')
```

### 틀린 이유
- 맨 첫줄에 불필요한 줄바꿈이 있어서, 빈 줄이 그림의 맨 윗줄에 추가되었음
